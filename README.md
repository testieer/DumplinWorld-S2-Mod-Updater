# Wondering how to use .ps1 files?
## Wishing that some random tutorial can help you?
### Well no wonder! This guide will help you how to use .ps1 files! (watching you urmy)
First of all, you should have the .ps1 file sitting in your computer downloaded FIRST okay?

Step 1, open up Run by doing Windows + R!

Step 2, type in "powershell" then press ok!

![STEP 2](https://i.postimg.cc/prc32TGq/Screenshot-2026-05-15-141709.png)

Step 3, On the terminal type in "Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process" to allow all .ps1 scripts!

![STEP 3](https://i.postimg.cc/Wz7grqLS/Screenshot-2026-05-15-142207.png)

Step 4, Go to File Explorer and go to the .ps1 file.

![STEP 4](https://i.postimg.cc/hvwg2pBh/Screenshot-2026-05-15-142600.png)

Step 5, Right click it and press "Copy as path"

![STEP 5](https://i.postimg.cc/N0fsfypF/Screenshot-2026-05-15-142837.png)

Step 6, Now go back to the terminal and type in "./{copied)". The {copied} part is the one you copied from step 5.

![STEP 6](https://i.postimg.cc/TwVC7wvm/Screenshot-2026-05-15-143058.png)

BONUS STEP, It might show up to allow scripts, so if it shows, press A.

Now you just ran a .ps1 file out from SCRATCH! YAY!
